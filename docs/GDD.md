# 미스테리 체이서: 검은 촛불 (팬메이드 리메이크) — 게임 기획 문서 (GDD)

> 2013년 한국 모바일 카드배틀 어드벤처 RPG **미스테리 체이서**에서 영감을 받은 팬메이드 게임.
> 원작의 상표·일러스트·텍스트는 사용하지 않으며, 시스템과 분위기를 재해석한 오리지널 콘텐츠로 제작한다.

---

## 1. 개요

| 항목 | 내용 |
|---|---|
| 제목 | 미스테리 체이서: 검은 촛불 (Mystery Chaser: The Black Candle) |
| 장르 | 스토리 중심 어드벤처 RPG + 카드배틀 |
| 플랫폼 | 웹 (모바일 브라우저 우선, 데스크톱 지원) |
| 기술 | HTML / CSS / Vanilla JS, localStorage 저장 |
| 타깃 | 원작 팬, 다크 고딕 스토리 게임 애호가 |
| 플레이 타임 | 프로토타입 기준 30~60분 (5개 챕터) |

### 원작에서 계승한 핵심 정체성
1. **카드는 덱 대결용이 아니라 "장비·도구"다** — 카드를 사용하면 주인공의 공격력·방어력·체력이 오르고, 그 능력치로 공방을 주고받는다.
2. **민첩성이 높은 쪽이 선공** — 매 라운드 턴 순서는 민첩(AGI) 비교로 결정.
3. **캐릭터별 고유 스킬, 전투당 1회** — "카드 2장 드로우", "상대 능력치 감소" 같은 강력한 일회성 스킬.
4. **어두운 도시 괴담 세계관** — 화려한 판타지가 아닌 19세기풍 뒷골목·교도소·공동묘지. 정신을 조종하는 흑마법, 인체실험 등 사회적 금기를 다루는 스토리.
5. **메인 스토리 + 캐릭터별 개인 사연** — 책을 읽듯 따라가는 서브 스토리.
6. **RPG식 성장** — 레벨업으로 능력치 상승, 진행할수록 더 좋은 카드 획득.
7. **카드 뽑기** — 원작의 과금 요소를 무과금 인게임 재화(골드)로 대체.

## 2. 세계관·스토리

### 배경
안개와 비에 젖은 19세기풍 항구 도시 **그레이헤이븐(Grayhaven)**.
최근 시민들이 하나둘 이유 없이 미쳐가고, 실종자가 늘어난다. 소문의 중심에는
사람의 정신을 잠식하는 흑마법을 연구하는 비밀 결사 **「검은 촛불」**이 있다.
주인공들은 각자의 이유로 이 사건을 쫓는 "체이서"가 된다.

### 메인 스토리 (프로토타입 5챕터)
| 챕터 | 무대 | 사건 | 보스 |
|---|---|---|---|
| 1. 안개 낀 부두 | 부둣가 뒷골목 | 광기에 사로잡힌 실종자 발견 | 광인이 된 부두 노동자 |
| 2. 검은 촛불 | 폐쇄된 예배당 | 결사의 의식 현장 급습 | 검은 촛불의 수도사 |
| 3. 지하 실험실 | 옛 교도소 지하 | 인체실험의 증거 발견 | 실험체 9호 |
| 4. 마녀의 밤 | 공동묘지 | 결사에 협력하는 마녀와 대면 | 마녀 헤카테의 그림자 |
| 5. 촛불이 꺼질 때 | 대성당 첨탑 | 흑막과의 최종 대결 | 주교 발타자르 |

### 서브 스토리
각 플레이어블 캐릭터는 개인 사연 1편을 보유. 허브 화면에서 언제든 열람 가능(책 형식 텍스트).

## 3. 캐릭터

> **규칙·수치의 단일 기준은 [FEATURE_SPEC.md](FEATURE_SPEC.md)** (§2 캐릭터 시스템). 여기서는 콘셉트만 요약한다.

원작의 캐릭터 유형(얼음을 다루는 귀공자 / 정신나간 신부 / 천재소녀 / 열혈기자)을 오리지널 캐릭터로 재해석.

| 캐릭터 | 유형 | 아키타입 | 고유 스킬 (전투당 1회) |
|---|---|---|---|
| 에드윈 프로스트 | 얼음의 귀공자 | 컨트롤 탱커 | **절대영도**: 적 공격력 30% 감소 (지속) |
| 그레고르 신부 | 광기의 사제 | 서스테인 탱커 | **광신의 기도**: 최대 HP의 40% 회복 |
| 아리아 벨 | 천재소녀 | 테크니컬 딜러 | **완전분석**: 카드 2장 드로우 + 다음 공격 관통 ×2 |
| 잭 하퍼 | 열혈기자 | 스피드 딜러 | **특종감각**: 3라운드 동안 공격력 ×1.5 |

- 뱀파이어·마녀 등 초자연 존재는 적/NPC로 등장 (추후 플레이어블 확장 여지)

## 4. 전투 시스템

> **상세 규칙(라운드 시퀀스, 회피, 흑마법, 종료 조건)과 수치는 [FEATURE_SPEC.md](FEATURE_SPEC.md) §3 참조.**

### 핵심 흐름 요약
1. 전투 시작 시 보유 카드 전체를 섞어 **산차(deck)** 구성, 3장을 손패로 드로우
2. 매 라운드: (선택) 카드 1장 사용 → (선택) 고유 스킬 → **공격 실행** (AGI 높은 쪽 선공) → 카드 1장 드로우
3. 어느 한쪽 HP가 0이 되면 종료. 민첩 우위는 **회피율**로도 이어진다
4. 적은 4라운드마다 방어 무시 **흑마법** 사용 — 원작의 "정신을 잠식하는 흑마법" 연출

### 카드 = 장비/도구
| 등급 | 색 | 예시 | 뽑기 확률 |
|---|---|---|---|
| 일반 | 회색 | 녹슨 단검(+3 ATK), 붕대(HP 15 회복) | 60% |
| 희귀 | 파랑 | 은장도(+6 ATK), 성수(HP 30 회복) | 25% |
| 영웅 | 보라 | 처형인의 대검(+10 ATK), 그림자 망토(+6 AGI) | 12% |
| 전설 | 금색 | 서리검 위스퍼(+15 ATK), 생명의 물약(HP 전체 회복) | 3% |

- 장비형 카드(무기/방어구/시계 등)의 버프는 **해당 전투 동안 지속**
- 소모품형 카드(붕대/성수/물약)는 즉시 효과
- 시작 카드: 녹슨 단검×2, 낡은 코트×1, 붕대×2

## 5. 성장·경제

- **골드**: 챕터 클리어 보상. 클리어한 챕터는 **재도전 가능**(반복 파밍, 보상 50%)
- **카드 뽑기**: 80골드/1회. 실제 과금 없음 — 원작 가챠의 "느긋하게 즐기면 소과금으로 충분" 철학을 무과금으로 재해석
- **첫 클리어 보너스**: 챕터별 고정 카드 1장 드롭

## 6. 화면 구성

```
타이틀 → 캐릭터 선택 → 허브
                         ├─ 스토리 (챕터 선택 → 스토리 씬 → 전투 → 결과)
                         ├─ 카드 뽑기
                         ├─ 카드 보관함
                         └─ 캐릭터 상태 / 서브 스토리
```

- 모바일 세로 화면(360px~) 우선 레이아웃
- 다크 고딕 팔레트: 잉크 블랙 배경 + 촛불 금색 포인트 + 등급별 카드 색
- 저장: localStorage 자동 저장 (챕터 클리어·뽑기·레벨업 시점)

## 7. 프로토타입 범위 (v0.1)

**포함**: 캐릭터 4인, 챕터 5개(스토리+전투), 카드 16종, 뽑기, 레벨업, 서브 스토리 4편, 자동 저장
**제외(추후)**: PvP 랭킹, 카드 강화/합성, 사운드, 일러스트(현재 이모지/타이포로 대체), 추가 챕터

## 8. 법적 고려
- 팬메이드 비상업 프로젝트. 원작의 에셋(일러스트·음원·텍스트)을 일절 사용하지 않음
- 캐릭터·지명·스토리는 원작 "유형"만 참고한 오리지널 창작물
- 배포 시 "팬메이드 게임이며 원작과 무관함"을 명시
