# 기능 기획서 (Feature Specification) — v1.0 원작 재현판

> **이 문서가 게임 규칙·수치·데이터의 단일 기준(Single Source of Truth)이다.**
> 구현이 이 문서와 다르면 구현이 버그다. 모든 수치는 `tools/balance-sim.js`(시행 5,000회)로 검증한다.
> v0.x(장비 버프형 전투)의 명세·검증 이력은 git 히스토리 참조. v1.0은 **원작 플레이 증언에 따라 전투 모델을 전면 교체**했다.

## 0. v1.0 전투 모델의 근거 (원작 재현)

원작 플레이어 증언 + 2013년 기사 기록으로 확정한 원작 전투:

| 확정 사항 | 출처 |
|---|---|
| **덱을 구성하고, 매 턴 카드 1장을 내서 그 카드로 공격** | 플레이어 증언 |
| **카드 자체에 데미지(공격)·회피 능력치** | 플레이어 증언 |
| 낸 카드는 버려지고 **덱이 순환** (소진 시 리셔플) | 플레이어 증언 |
| **캐릭터 HP 단일** (카드는 파괴되지 않음) | 플레이어 증언 |
| **특정 카드를 집중 강화**해 에이스로 육성 | 플레이어 증언 |
| 민첩 높은 쪽 선공 / 캐릭터별 전투당 1회 고유 스킬 | 기사 기록 |
| 마녀 캐릭터 스킬 = **적 민첩 대폭 감소로 선공 탈취** | 기사 기록 |
| 레벨업 시 **전투 지참 카드 수 증가** | 기사 기록 |
| 과금(→인게임 골드)은 **카드 뽑기 + 소모품 구매** | 기사 기록 |

## 1. 코어 루프

```
허브 ─ 덱 편성(지참 한도 = 4+레벨) ─▶ 전투(챕터/던전/결투)
  │        ▲                            │
  │        └── 카드 뽑기(골드) ──────────┤ 보상: EXP·골드·카드
  │        └── 에이스 카드 강화(골드) ───┤
  │        └── 소모품 구매(상점) ────────┘
성장 축 3개: ① 레벨(HP/민첩/공격 보정/덱 한도) ② 덱 구성 ③ 에이스 카드 강화
```

## 2. 캐릭터

### 2.1 스탯
| 스탯 | 역할 |
|---|---|
| HP | 0이 되면 패배. 레벨당 +8 |
| 민첩 AGI | ① 선공 판정(동률=플레이어) ② 2레벨당 +1 |
| 공격 보정 | 낸 카드의 공격에 더해짐 = `round(기본공격/2) + (레벨-1)` |

- 필요 경험치 = 레벨 × 40 (v0.x와 동일 곡선 — 챕터 N을 레벨 N에 도전)
- **덱 지참 한도 = 4 + 레벨 (최대 12장)** — 원작의 "레벨업 시 카드 더 지참"

## 3. 전투 시스템 (덱 순환식)

### 3.1 라운드 시퀀스
```
[전투 시작] 양측 덱 셔플 → 각 3장 드로우
[매 라운드]
  1. 플레이어: 손패에서 카드 1장 선택 (필수) — 이 카드가 이번 라운드의 공격·회피
     (선택) 소모품 1개 사용 / (선택) 고유 스킬 (전투당 1회)
  2. 적/고스트: AI가 카드 선택 (평소 최고 공격, HP 40% 미만이면 최고 회피)
     고스트는 고유 스킬도 사용 (캐릭터별 고정 타이밍)
  3. 격돌: AGI 높은 쪽 선공 (동률=플레이어) → 생존 시 반격
  4. 낸 카드는 버림 더미로, 양측 1장 드로우. 산차 소진 시 버림 더미 리셔플 (덱 순환)
[종료] HP ≤ 0 / 도주 / 30라운드 초과(무승부=패배 취급, 보상 없음)
```

### 3.2 데미지·회피 공식
```
공격력 = 카드 공격(강화 포함) + 공격 보정
데미지 = max(1, round(공격력 × 스킬계수 × U(0.9, 1.1)))
회피   = 수비측이 이번 라운드에 낸 카드의 회피%(강화 포함) 판정. 성공 시 데미지 0
```
- 스킬계수: 특종감각 ×1.5(3R) / 절대영도 피해자 ×0.7 — 합산이 아닌 곱연산
- **흑마법** (스토리 적 전용): 5라운드마다 — 회피 불가, ×1.2
- 손패 최대 5장. 도주 버튼·결과 처리(FR-22)는 v0.6과 동일

- 중복 카드는 덱에 여러 장 채용 가능

### 4.2 카드 강화 (에이스 육성 — 원작 핵심)
- 카드 종류 단위 **+1~+5**: +1당 ⚔ +2, 👟 +2%p (회피 상한 45%)
- 비용: `(현재강화+1) × 30 × 등급계수(일반1/희귀2/영웅3/전설5)` 골드 — **중복 카드 소모 없음** (골드만)
- 예: 은장도 +5까지 총 900G, 위스퍼 +5까지 총 2,250G
- 검증: 5장 보스 기준 에이스(처형인 대검) +5 시 승률 +25%p (§7.2)

### 4.3 덱 편성
- 보유 카드에서 지참 한도(4+레벨, 최대 12)만큼 선택. 자동 편성(공격+회피 상위) 제공
- 덱이 비면 전투 시작 시 자동 편성으로 보정. 보유 수량 초과·미지 카드는 자동 정리

### 4.4 소모품 (카드와 별개 — 원작의 "소모품 구매")
| 이름 | 효과 | 가격 |
|---|---|---|
| 🩹 붕대 | HP 30 회복 | 30G |
| 💧 성수 | HP 60 회복 | 70G |
| 🧪 비약 | HP 120 회복 | 150G |
| 💖 생명의 물약 | 전체 회복 | 300G |
| ☕ 각성제 | 민첩 +3 (전투 지속) | 50G |

- 전투 중 **라운드당 1개** 사용. **사용 즉시 영구 소모** (패배해도 반환 없음)
- 상점(뽑기 화면)에서 구매. 시작 지급: 붕대×2. 던전 드롭 30%로 붕대 수급
