# 미스테리 체이서: 검은 촛불 (팬메이드 리메이크)

2013년 한국 모바일 카드배틀 어드벤처 RPG **미스테리 체이서**에서 영감을 받은 비상업 팬메이드 게임입니다.
원작의 에셋(일러스트·음원·텍스트)은 사용하지 않으며, 시스템과 분위기를 재해석한 오리지널 콘텐츠로 제작했습니다.

## 플레이 방법

별도 빌드 없이 브라우저에서 바로 실행됩니다.

```bash
# 방법 1: 파일 직접 열기
index.html 을 브라우저로 열기

# 방법 2: 로컬 서버 (권장)
npx serve .
# 또는
python3 -m http.server 8000
```

모바일 세로 화면에 최적화되어 있으며, 진행 상황은 브라우저 localStorage에 자동 저장됩니다.

## 게임 특징 (원작 시스템 재해석)

- **카드 = 장비/도구**: 덱 대결이 아니라, 카드를 사용해 공격력·방어력·체력을 올리고 그 능력치로 공방을 주고받는 원작 고유의 전투 방식
- **민첩 기반 턴 순서**: 매 라운드 민첩이 높은 쪽이 선공
- **고유 스킬 (전투당 1회)**: 카드 2장 드로우, 적 공격력 감소 등 캐릭터별 일회성 스킬
- **다크 고딕 세계관**: 19세기풍 항구 도시의 뒷골목·교도소·공동묘지, 정신을 잠식하는 흑마법과 인체실험을 쫓는 스토리
- **캐릭터 4인**: 얼음의 귀공자 / 광기의 사제 / 천재소녀 / 열혈기자 — 각자 개인 사연(서브 스토리) 보유
- **성장·수집**: 레벨업, 챕터 5개, 카드 16종(4등급), 인게임 골드 카드 뽑기 (실제 과금 없음)
